## 🎬 Анимации в действии

### Загрузка страницы

```
0.0s   Hero появляется (fade in)
       ├─ Логотип вращается
       └─ Фон пульсирует

0.1s   Поиск поднимается снизу

0.2s   Категории появляются слева

0.3s   Статистика появляется

0.4s   Недавние приложения
       └─ Каждая карточка с задержкой 50ms

0.5s   Featured секция
       └─ Stagger анимация карточек

0.6s   More Apps секция

0.7s   Footer появляется
```

### Наведение на карточку

```
0ms     Масштаб: 1.0 → 1.02
        Тень: sm → xl
        Подъем: 0 → -4px

100ms   Эффект свечения (glow)

200ms   Кнопка "Launch" появляется

300ms   Готово к клику
```

### Клик по карточке

```
0ms     Масштаб: 1.02 → 0.98 (feedback)

150ms   Переход к приложению

300ms   Анимация перехода страницы
```
